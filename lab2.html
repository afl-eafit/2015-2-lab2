Introduction

The aim of this programming lab is to write a Haskell program that filters a CVS dataset using regular expressions.

Assignment (80%)

  • (70%) Write a haskell program hfilter that given a dataset (in CSV format) and n regular expressions, filters out any row whose ith column does not match the ith regular expression completely, with 0 ≤ i ≤ n
  • (10%) Annotate your source code using Haddock. Documentation should explain your solution for the assignment. Improve it with HLint.
  • (Optional) Package your Haskell program with Cabal.

Example (Input and output)

With data.csv as dataset, and 3 regular expressions 

      $ cat data.csv

      Alejandro,Gomez,123
      Andrés,sicard,765
      pepito,Perez,8765

      $ hfilter data.csv '[A-Z][A-Za-z]*' '[A-Z][A-Za-z]*' '[0-9]\{,3\}'

      Alejandro,Gomez,123

Note that 2 of 3 lines were filtered out for mismatches of 1 or more columns with their respective regular expression.

Clean code (10%)

Before submitting your code, clean it up! Clean code:

  • Does not have long lines (at most 80 columns).
  • Has a consistent layout.
  • Has type signatures for everything (included where and let definitions).
  • Has good comments.
  • Has no junk (unused code, commented code, unnecessary code).
  • Has no overly complicated function definitions.
  • Does not contain any repetitive code.
  • Has no tabs.
  • Has no unnecessary spaces at the end of a line, or empty lines at the end of a file. (See: Fixing trailing whitespace issues with Emacs ).
